Multi-Owner Etheruem Wallet

An Ethereum wallet that allows multiple addresses acts as joint owners. The wallet has an administrative address with the permision to add or delete an owner address. All the owners have the permission to propose a transaction, which would then require the approval of other owners before the transaction is executed. At deployement, a minimum of 60% approval is required for a transaction to be executed, with the administrator address having the permission to changge the required percentage of approvals.
